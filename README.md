# Консольный индексатор файлов

Учебный проект (полный вариант). Индексирует указанную папку и сохраняет метаданные о файлах в базу данных SQLite для последующего анализа и поиска дубликатов.

## Структура проекта
- `src/` — исходный код программы.
- `src/main.py` — точка входа, CLI интерфейс.
- `src/db.py` — модуль для инициализации и работы с базой данных SQLite.
- `data/` — папка для хранения базы данных (создается автоматически).

## Требования
- Python 3.x (используется только стандартная библиотека)

## Запуск
```bash
python src/main.py <путь_к_папке>
